# cookie-cats-ab-test
Cookie Cats A/B Test Analysis

Goal:
Evaluate whether moving the first gate from level 30 to level 40 increases player engagement.

Dataset:
Kaggle [Cookie Cats A/B test dataset](https://www.kaggle.com/datasets/zahrazolghadr/ab-test-cookie-cats/data)

Methods:
- experiment design (MDE, power analysis)
- descriptive statistics
- Welch t-test

Result:
No statistically or practically significant difference in number of game rounds played.
